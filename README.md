# Sistema de Quiosque de Autoatendimento

Projeto final da disciplina de Programação Orientada a Objetos.
Simula um totem de fast-food com interface gráfica e módulo de cozinha.

## Funcionalidades
- Cliente: Seleção de lanches/bebidas com fotos, carrinho de compras dinâmico (adicionar/remover) e finalização de pedido.
- Cozinha: Monitoramento de pedidos em tempo real com alteração de status (Em Preparo -> Pronto -> Entregue).
- Segurança: Acesso restrito à cozinha via senha administrativa.

## Tecnologias Utilizadas
- Java (JDK 17+)
- JavaFX (Interface Gráfica)
- Maven (Gerenciamento de dependências)
- VS Code (IDE Recomendada)

## Como Rodar o Projeto
1. Certifique-se de ter o Java e o Maven instalados.
2. Abra a pasta do projeto no VS Code.
3. Aguarde o Maven baixar as dependências.
4. Abra o arquivo `src/main/java/com/trabalhoprog1/App.java`.
5. Pressione F5 ou clique em "Run".
